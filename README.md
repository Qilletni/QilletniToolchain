# Qilletni Toolchain

The Qilletni toolchain is the CLI for running, building and documenting Qilletni code, installed through the `qilletni` commmand. This is the sold entrypoint for the main [Qilletni](https://github.com/Qilletni/Qilletni) language implementation.

For usage, see the [qilletni.dev](https://qilletni.dev/) website.
